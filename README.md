Ce projet c'est un système de gestion scolaire . Il a pour but de digitaliser l'ancien système scolaire et de faciliter le travail de chaque cadre dans un établissement.



ce projet a été conçu par:

\-Le langage de programmation PHP

\-HTML \& CSS

\-Base de données relationnelle MySQL

\-serveur locale durant le développement : Wamp64   



Avec ce système de gestion scolaire numérise, on a la possibilité de ce connecté sur 4 rôles différents :

\-Admin

\-Secrétaire/Secrétaire général

\-Comptable 

\-Enseignant



L'admin a acces a tous les role dans ce systeme : la creation d'une classe , l'ajout d'un enseignant, suivi en temps reel de tous les actions dans l'etablissement

Le Secretaire/SG peut ajouter, lister les sanctions/Absences et retard d'un eleve, consulter les moyennes d'un eleve, d'une classe et generer un bulletin pret a l'impression

Le comptable  gere tous les finances d'une etablissement

et enfin l'enseignant qui verifier l'assiduite/le profil et les moyennes, il peut ajouter une note selon sa matiere et le systeme aditionne automatiquement les notes pour afficher les resultat selon les coefficient 

&#x20;ce systeme a pour but de faciliter de rendre l'education a Madagascar Numerique pour diminuer les risques d'erreur sur papier
